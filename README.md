# Conflict Testing Repository

This repository is designed to test EVERY possible merge conflict scenario.

## Test Scenarios Included:

1. Simple line conflicts
2. Multi-line conflicts
3. Function signature conflicts
4. Class definition conflicts
5. Import statement conflicts
6. JSON conflicts
7. Configuration file conflicts
8. Whitespace conflicts
9. Indentation conflicts
10. Comment conflicts
11. Complex nested conflicts
12. Binary file conflicts
13. Large file conflicts
14. Multiple conflicts in single file
15. Cross-file dependency conflicts

## Branches:

- `main` - Base branch
- `feature-a` - Feature branch A
- `feature-b` - Feature branch B (conflicts with A)
- `refactor` - Major refactoring (conflicts everywhere)
- `hotfix` - Emergency fixes (conflicts with features)
